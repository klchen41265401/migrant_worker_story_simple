E 與移工（移工）認識經過的真實紀實網站，記錄 2024 年 1 月至 2026 年 4 月間的故事。

## 技術細節

- **模組化靜態網站**：`index.html` 為骨架，內容由 `sections/` 下的 HTML 片段組成
- **動態載入**：`js/main.js` 使用 `fetch()` 平行載入所有區段，GitHub Pages 完全相容
- **CSS 外部引用**：Google Fonts (Noto Sans TC)、CSS 變數主題色、時間軸排版
- **RWD 響應式**：支援手機與桌面瀏覽
- **功能**：頂部進度條、黏性導覽列、回到頂部按鈕、滾動追蹤高亮、可收合區塊（details/summary）
- **注意**：因使用 `fetch()` 載入片段，本機需使用 HTTP 伺服器瀏覽（如 `python -m http.server`），不支援直接用 `file://` 開啟
